
This command is simply called **forward** followed by a number. The number indicates how far the turtle moves forward.

The turn orders are **left** or **right** followed by a number. The words "left" and "right" tell the turtle which side to turn to.

Before starting a new drawing you can always clear the screen with the following command **clearscreen**

We can use shortcuts to some commands. Instead of "right" it's enough to write **rt** . Instead of "left" write **lt** , instead of "forward" write **fd**, and instead of "clearscreen" **cs**
